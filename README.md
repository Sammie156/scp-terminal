## Usage : 

Currently working commands include : 
```ps
access scp-2021
```
This would access details on SCP-2021 and display the details on the screen as shown below : 
![image](https://user-images.githubusercontent.com/64217426/142717013-e6e158eb-a0ab-46c0-820d-052e5cd5d267.png)

```ps
access random
```
This accesses a random SCP.
